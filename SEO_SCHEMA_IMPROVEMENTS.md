- Add FAQ section to answer common questions
- Hours, menu questions, catering inquiries

Example:
```json
{
    "@type": "FAQPage",
    "mainEntity": [
        {
            "@type": "Question",
            "name": "What are your hours?",
            "acceptedAnswer": {
                "@type": "Answer",
                "text": "Open daily 11 AM. Wed-Thu close at 9 PM, Fri-Sat close at 9:30 PM"
            }
        }
    ]
}
```
